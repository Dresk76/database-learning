# Database Learning

Repositorio dedicado a mi aprendizaje y práctica de **bases de datos** y tecnologías relacionadas con el almacenamiento, consulta y manipulación de datos.

Este proyecto organiza **ejercicios, ejemplos y notas** mientras estudio diferentes conceptos de bases de datos y SQL.

## Tecnologías

Actualmente este repositorio incluye prácticas con:

* SQL
* MySQL
* phpMyAdmin

En el futuro se agregarán otras tecnologías como:

* PostgreSQL
* MongoDB
* Redis

### Descripción

**mysql/**
Contiene todo el material relacionado con el aprendizaje de MySQL.

* **DDL.sql**
  Comandos de **Data Definition Language**, utilizados para definir estructuras de bases de datos (CREATE, ALTER, DROP, TRUNCATE).

* **DML.sql**
  Comandos de **Data Manipulation Language**, utilizados para insertar y consultar datos (INSERT, SELECT, etc.).

* **queries/**
  Consultas SQL organizadas por temas y ejemplos de uso.

* **exercises/**
  Ejercicios prácticos para aplicar los conceptos aprendidos.

## Objetivos de Aprendizaje

* Comprender los fundamentos de las **bases de datos relacionales**.
* Practicar la escritura de **consultas SQL**.
* Diseñar **tablas y estructuras de datos**.
* Trabajar con operaciones **CRUD**.
* Organizar proyectos relacionados con bases de datos.
